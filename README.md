
### Build Setup

``` bash
这里建议用cnpm

#### install dependencies
npm install

#### 启动网页多页应用
npm run dev

#### 启动网页单页应用
npm run dev:wx

#### 打包生产环境代码
npm run build:wx

```

### 目录说明
hbbuild 工程化层<br>
src 项目代码目录<br>
-- assets 静态文件图片存放目录<br>
-- mocks 模拟数据录入<br>
-- modules  业务模块存放目录<br>
-- pages 应用入口目录<br>
apis.js 应用后端api配置<br>
config.js 应用基础配置<br>
routes.js 单页路由配置<br>

For detailed explanation on how things work, checkout the [guide](http://vuejs-templates.github.io/webpack/) and [docs for vue-loader](http://vuejs.github.io/vue-loader).
